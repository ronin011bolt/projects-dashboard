# Source Evaluation

Use this reference when judging how much weight to give a source.

## Primary questions

- Is this a primary or secondary source?
- Is the source direct, copied, summarized, or quoted out of context?
- Is there a date or timestamp?
- Is there a motive, bias, or strategic framing?
- Can any part be independently corroborated?

## Reliability cues

Stronger signals:
- original document or first-party post
- official filing or direct record
- contemporaneous timestamp
- corroboration from independent sources
- stable archival trace

Weaker signals:
- unattributed screenshot
- repost chain
- anonymous summary without evidence
- selective excerpt with no context
- unverifiable translation or paraphrase

## Suggested labels

- high confidence
- medium confidence
- low confidence
- unknown confidence

Confidence is not truth. It is a working judgment based on available evidence.
